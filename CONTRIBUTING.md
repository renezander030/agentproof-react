# Contributing

AgentProof React is intentionally small: deterministic gates for AI-generated React and Next.js code.

Good contributions usually add one of three things:

1. A high-signal rule for a common AI-generated frontend failure.
2. A better proof moment in the demo fixtures.
3. CI, JSON, SARIF, or PR-comment output improvements.

Please include a bad fixture and a good fixture when proposing a new rule.
